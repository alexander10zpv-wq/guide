# Задача урока 2

Открой [task.py](task.py) и выполни шаги, отмеченные `//TODO`.

## Что нужно сделать

1. Создай переменные `is_admin = True` и `is_blocked = False`.
2. Реализуй функцию `can_login(is_active, is_blocked)`, которая возвращает `True`, если пользователь активен и **не** заблокирован, иначе `False`.
3. Реализуй функцию `is_adult(age)`, которая возвращает `True`, если `age >= 18`, иначе `False`.
4. Реализуй функцию `is_empty_name(name)`, которая возвращает `True`, если строка `name` пустая или состоит только из пробелов.
   - Подсказка: `name.strip()` уберёт пробелы по краям. Пустая строка приводится к `False`.

## Как запускать и проверять

1. Открой терминал в VSCode (`Ctrl + ~`).
2. Перейди в папку:
   ```powershell
   cd lesson_02_bool
   ```
3. Запусти проверку:
   ```powershell
   python check.py
   ```
