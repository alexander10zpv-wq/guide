## Что нужно сделать

1. Реализуй `process_queue(tasks)` — принимает список задач. Создай `deque`, добавь туда все задачи, потом по очереди извлекай **с начала** через `popleft` и собирай результат в список. Верни список в порядке обработки.
2. Реализуй `top_words(words, n)` — принимает список слов. Вернуть список из `n` самых частых слов **(только слова, без счётчика)**. Используй `Counter.most_common`.
3. Реализуй `group_by_first_letter(words)` — принимает список строк. Вернуть `dict` (можно `defaultdict`, потом приведи к `dict`), где ключ — первая буква, значение — список слов с этой буквы.
4. Создай `namedtuple` `Point` с полями `x, y`. И функцию `make_point(x, y)`, которая возвращает `Point(x, y)`.
5. Реализуй `count_letters(text)` — вернуть `dict` буква → количество. Используй `defaultdict(int)`, потом приведи к `dict`.
