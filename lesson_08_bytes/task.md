# Задача урока 8

Открой [task.py](task.py) и выполни все `//TODO`.

## Что нужно сделать

1. Реализуй `to_bytes(text)` — закодируй строку в `bytes` через UTF-8.
2. Реализуй `from_bytes(data)` — раскодируй `bytes` обратно в строку через UTF-8.
3. Реализуй `replace_first_byte(data, new_byte)` — принимает `bytes`, возвращает **новый** `bytes`, где первый байт заменён на `new_byte`. Преобразуй в `bytearray`, поменяй, верни `bytes`.
4. Реализуй `bytes_length(data)` — вернуть число байтов в `bytes`-объекте. Подумай, почему длина в байтах для русского текста больше длины в символах.
5. Реализуй `is_ascii(data)` — вернуть `True`, если все байты в `data` < 128.
