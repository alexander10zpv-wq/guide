# Урок 1. Введение в типы данных. Числа `int`, `float`, `complex`

## Что такое тип данных

Тип данных определяет, **какие значения** может хранить объект и **какие операции** с ним можно выполнять. В Python всё является объектом — даже число, строка или функция.

Узнать тип можно через `type()`:

```python
x = 10
print(type(x))  # <class 'int'>
```

## Числовые типы

### `int` — целые числа

```python
age = 25
count = 100
user_id = 123456
```

**Когда использовать:**
- ID пользователей, счётчики, индексы, количество товаров, возраст.
- Деньги в минимальных единицах (копейки, центы): `price_cents = 1999`.

**Преимущества `int`:**
- Нет ошибок округления.
- Поддерживает очень большие числа.
- Удобен в индексах и циклах.

### `float` — числа с плавающей точкой

```python
temperature = 36.6
percent = 12.5
distance = 10.75
```

**Когда использовать:** температура, расстояния, проценты, координаты, научные расчёты.

**Когда НЕ использовать:** деньги, бухгалтерия. Причина:

```python
print(0.1 + 0.2)  # 0.30000000000000004
```

`float` хранится в бинарном виде и не всегда точно представляет десятичные дроби.

### `complex` — комплексные числа

```python
z = 3 + 4j
```

В обычной разработке нужны редко (математика, физика, обработка сигналов).

## Базовые операции

```python
a = 10
b = 3

print(a + b)   # 13     сложение
print(a - b)   # 7      вычитание
print(a * b)   # 30     умножение
print(a / b)   # 3.333  деление (всегда float!)
print(a // b)  # 3      целочисленное деление
print(a % b)   # 1      остаток
print(a ** b)  # 1000   возведение в степень
```
