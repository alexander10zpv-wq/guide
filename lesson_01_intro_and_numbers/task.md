# Задача урока 1

Открой [task.py](task.py) и выполни шаги, отмеченные `//TODO`.

## Что нужно сделать

1. Создай переменную `age` типа `int` со значением `25`.
2. Создай переменную `price` типа `float` со значением `19.99`.
3. Создай переменную `z` типа `complex`, равную `3 + 4j`.
4. Реализуй функцию `rectangle_area(width, height)`, которая возвращает площадь прямоугольника. Аргументы — целые или дробные числа.
5. Реализуй функцию `cents_to_dollars(cents)`, которая принимает целое число центов и возвращает строку вида `"12.34"` (доллары и центы через точку).
   - Подсказка: используй целочисленное деление `//` и остаток `%`.

## Как запускать и проверять

1. Открой терминал в VSCode: **Terminal → New Terminal** (или `Ctrl + ~`).
2. Перейди в папку урока:
   ```powershell
   cd lesson_01_intro_and_numbers
   ```
3. Запусти проверку:
   ```powershell
   python check.py
   ```
4. Если все проверки прошли — увидишь сообщение `Все задачи выполнены верно!`.
   Если есть ошибки — внимательно прочитай вывод и поправь `task.py`.
