# Финальная контрольная

Это **итоговая** работа по всему курсу. Здесь нет новой теории — вернись к материалам уроков 1-17, если что-то забыл.

## Что проверяется

| Урок | Тема | Применение в этой задаче |
|------|------|--------------------------|
| 1    | `int`, `float`        | количество, размеры |
| 2    | `bool`                | флаги, проверки |
| 3    | `str`                 | email, форматирование |
| 4-5  | `list`, `tuple`       | список товаров |
| 6    | `dict`                | словарь товаров и счётчики |
| 7    | `set`                 | уникальные категории |
| 8    | `bytes`               | сериализация в байты |
| 9    | `None`                | «не найдено» |
| 10   | `Decimal`             | цены, точные суммы |
| 11   | `datetime`            | момент создания заказа |
| 12   | `Enum`                | статусы заказа |
| 13   | `pathlib.Path`        | сохранение / чтение файла |
| 14   | `Counter`, `deque`    | подсчёт, очередь обработки |
| 15   | `dataclass`           | модели `Product`, `Order` |
| 16   | `typing`              | аннотации типов |
| 17   | выбор типа            | весь дизайн API |

См. [task.md](task.md) и [task.py](task.py).
