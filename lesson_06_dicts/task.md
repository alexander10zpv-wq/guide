# Задача урока 6

Открой [task.py](task.py) и выполни все `//TODO`.

## Что нужно сделать

1. Создай словарь `user` с ключами `"name": "Alex"`, `"age": 25`, `"email": "alex@example.com"`.
2. Реализуй `get_field(data, key)` — безопасно вернуть значение по ключу (если ключа нет — `None`). Используй `.get()`.
3. Реализуй `set_field(data, key, value)` — установить ключ и вернуть обновлённый словарь.
4. Реализуй `merge_dicts(a, b)` — вернуть новый словарь, объединяющий `a` и `b`. При совпадении ключей побеждают значения из `b`. Исходные словари не менять.
5. Реализуй `count_chars(text)` — вернуть словарь `{буква: количество}` для каждой буквы строки (пробелы тоже учитываются).
6. Реализуй `invert_dict(data)` — вернуть словарь, где значения и ключи поменяны местами. Предполагай, что значения уникальны и hashable.

## Как запускать и проверять

2. `cd lesson_06_dicts`
3. `python check.py`
