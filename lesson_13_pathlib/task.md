# Задача урока 13

Открой [task.py](task.py) и выполни все `//TODO`.

## Что нужно сделать

1. Реализуй `build_path(parent, name)` — принимает строку `parent` и строку `name`. Вернуть **`Path`**, равный `parent / name`.
2. Реализуй `get_extension(path)` — принимает `Path`. Вернуть расширение файла (включая точку): `.py`, `.txt`, ...
3. Реализуй `change_extension(path, new_ext)` — заменить расширение. `new_ext` начинается с точки, например `".md"`. Вернуть новый `Path`.
4. Реализуй `write_and_read(path, text)` — записать `text` в файл, потом прочитать и вернуть содержимое. Используй UTF-8.
5. Реализуй `safe_read(path)` — вернуть содержимое файла строкой или пустую строку `""`, если файла нет (без падения).

## Как запускать и проверять

1. Открой терминал (`Ctrl + ~`).
2. `cd lesson_13_pathlib`
3. `python check.py`

> Проверка создаёт временные файлы во временной папке системы — рабочий каталог не засоряется.
