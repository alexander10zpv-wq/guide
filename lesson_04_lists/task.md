## Что нужно сделать

1. Создай список `fruits = ["apple", "banana", "cherry"]`.
2. Реализуй `add_item(items, value)` — добавляет `value` в конец списка `items` и возвращает обновлённый список.
3. Реализуй `remove_item(items, value)` — удаляет первое вхождение `value`. Если значения нет — вернуть список без изменений (используй `try/except`).
4. Реализуй `sum_positive(numbers)` — возвращает сумму положительных чисел из списка.
5. Реализуй `unique_sorted(items)` — возвращает новый отсортированный список без дублей.
6. Реализуй `squares_of_evens(n)` — возвращает список квадратов чётных чисел от 0 до `n-1` (используй list comprehension).
