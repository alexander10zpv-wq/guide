# Задача урока 3

Открой [task.py](task.py) и выполни все `//TODO`.

## Что нужно сделать

1. Реализуй `normalize_email(email)` — возвращает email в нижнем регистре без пробелов по краям.
2. Реализуй `greet(name)` — возвращает строку `"Привет, <name>!"` (используй f-строку).
3. Реализуй `count_words(text)` — возвращает число слов в строке (по пробелам).
   - Подсказка: `text.split()` без аргументов делит по любым пробелам и убирает пустые.
4. Реализуй `is_palindrome(text)` — возвращает `True`, если строка читается одинаково в обе стороны. Игнорируй регистр и пробелы.
5. Реализуй `mask_card(card_number)` — принимает строку из 16 цифр и возвращает её с замаскированными цифрами в середине: `"1234123412341234" -> "1234********1234"`.
