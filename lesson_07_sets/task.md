## Что нужно сделать

1. Реализуй `unique(items)` — вернуть **множество** уникальных элементов из списка.
2. Реализуй `common(a, b)` — вернуть множество элементов, встречающихся одновременно в `a` и `b` (пересечение).
3. Реализуй `union_all(lists)` — вернуть объединение элементов из списка списков. Пример: `[[1,2],[2,3],[3,4]] -> {1,2,3,4}`.
4. Реализуй `only_in_first(a, b)` — вернуть элементы, которые есть в `a`, но нет в `b` (разность).
5. Реализуй `has_duplicates(items)` — вернуть `True`, если в списке есть хотя бы один дубль.
   - Подсказка: сравни `len(items)` и `len(set(items))`.
