# Урок 5. Кортежи `tuple` и диапазоны `range`

## `tuple` — неизменяемая упорядоченная коллекция

```python
point = (10, 20)
rgb = (255, 100, 50)
empty = ()
single = (42,)   # запятая обязательна для кортежа из одного элемента
```

## Когда использовать `tuple`

- Координаты, RGB-цвета.
- Фиксированные наборы данных, которые не должны меняться.
- Возврат нескольких значений из функции.
- Ключи словаря (списки в качестве ключей запрещены).

```python
def get_user():
    return ("Alex", 25)

name, age = get_user()  # распаковка
```

## Чем `tuple` отличается от `list`

| `list`                      | `tuple`                              |
|-----------------------------|--------------------------------------|
| изменяемый                  | неизменяемый                         |
| `[1, 2, 3]`                 | `(1, 2, 3)`                          |
| много методов (`append`, …) | только `count`, `index`              |
| не может быть ключом dict   | может быть ключом dict (если внутри hashable) |

```python
t = (1, 2, 3)
t[0]         # 1
len(t)       # 3
2 in t       # True
```

Изменение запрещено:

```python
t[0] = 99    # TypeError!
```

## `range` — генератор последовательности чисел

```python
range(5)         # 0, 1, 2, 3, 4
range(2, 7)      # 2, 3, 4, 5, 6
range(0, 10, 2)  # 0, 2, 4, 6, 8  (с шагом 2)
range(5, 0, -1)  # 5, 4, 3, 2, 1  (в обратную сторону)
```

```python
for i in range(5):
    print(i)
```

`range` **не создаёт** список чисел в памяти — числа генерируются по мере необходимости. Поэтому `range(1_000_000)` почти не занимает память.

Если нужен реальный список — приведи явно:

```python
list(range(5))   # [0, 1, 2, 3, 4]
```
