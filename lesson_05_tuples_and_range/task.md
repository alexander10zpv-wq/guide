3. Реализуй `min_max(numbers)` — возвращает кортеж `(минимум, максимум)` из списка. Для пустого списка — `(None, None)`.

5. Реализуй `countdown(n)` — возвращает список чисел от `n` до `1` (используй `range` с шагом `-1`).
